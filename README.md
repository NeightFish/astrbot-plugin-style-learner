# AstrBot Style Learner

**从对话记录学习说话风格，让 AI 模仿任何人说话。**

> 本插件由 AI (DeepSeek-V4-Pro) 生成。

## 功能

- 从 `.jsonl` 对话文件自动分析并提取说话风格
- **持续扮演**：开启后 bot 持续以指定风格回复，无需每次手动 `/imitate`
- **主动学习**：在日常聊天中自动积累对话数据，无需手动导入文件
- **智能示例检索**：jieba + TF-IDF 自动匹配最相关的对话示例，提升模仿质量
- 支持风格混合（多个风格按权重组合）
- 消息过滤：防止敏感信息污染数据集
- 使用统计与排行
- 技能重命名与合并
- 所有配置通过 AstrBot WebUI 暴露，开箱即用

## 安装

```bash
cd AstrBot/data/plugins
git clone https://github.com/NeightFish/astrbot-plugin-style-learner.git
cd astrbot-plugin-style-learner
pip install -r requirements.txt
```

2. 重启 AstrBot，或在 WebUI 中加载插件。

3. 在 WebUI → 插件配置中，填写 **DeepSeek API Key**（必填）。

### 持续扮演

开启后，bot 在该会话中会持续以指定风格回复，无需每次输入 `/imitate`：

```bash
/imitate 温柔学姐 --on        # 开启：后续消息自动以温柔学姐风格回复
/imitate --status             # 查看当前扮演状态
/imitate --off                # 关闭
```

也可以设**默认风格**（WebUI → `默认风格`），让 bot 对所有消息自动应用该风格。

### 风格混合

`/imitate` 支持多风格按权重混合：

```
/imitate 温柔学姐:0.3+沙雕群友:0.7 今天天气真好
```

### 技能合并

```
/skmerge 温柔学姐 沙雕群友 混合风格
```

## 主动学习

开启后，插件会在日常聊天中自动积累对话数据，达到阈值后自动分析并创建技能。

### 三种模式

| 模式 | 说明 |
|------|------|
| `off` (关闭) | 不做主动学习 |
| `all` (学习所有人) | 收集所有用户消息，创建技能 `@全局` |
| `specific` (指定用户) | 仅收集指定 QQ 号的消息，创建技能 `@<QQ号>` |

## 数据格式

对话数据为 `.jsonl` 文件（也支持 `.jsonl.gz`），每行一个 JSON 数组：

```json
["你好呀", "好久不见！最近过得怎么样？"]
["今天天气真好", "是呀，适合出去走走。"]
```

## 配置项

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `deepseek_api_key` | (空) | DeepSeek API Key，必填 |
| `deepseek_model` | `deepseek-chat` | 使用的模型名称 |
| `deepseek_base_url` | `https://api.deepseek.com` | API 地址 |
| `max_sample_chars` | `3000` | 学习时单次最大采样字符数 |
| `default_skill` | (空) | 默认风格技能名，填写后自动应用 |
| `active_learning_mode` | `off` | 主动学习模式: off / all / specific |
| `specific_qq` | (空) | specific 模式下指定的 QQ 号 |
| `batch_size` | `30` | buffer 满多少条后自动分析 |
| `enable_message_filter` | `true` | 是否启用关键词过滤 |
| `filter_keywords` | (预置) | 过滤关键词，逗号分隔 |
| `filter_prompt` | (空) | 语义审核 prompt（留空关闭） |
| `show_filter_notice` | `false` | 是否显示隐私提示 |
