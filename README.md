# AstrBot Style Learner

**从对话记录中学习说话风格，让 AI 模仿任何人说话。**

> 本插件由 AI (DeepSeek-V4-Pro) 生成。

## 功能

- 从 `.jsonl` 对话文件自动分析并提取说话风格
- 支持风格混合（多个风格按权重组合）
- 自动记录模仿输出到反馈日志，方便后续优化
- 所有配置通过 AstrBot WebUI 暴露，开箱即用

## 安装

1. 将本仓库克隆到 AstrBot 的插件目录：

```bash
cd AstrBot/data/plugins
git clone https://github.com/NeightFish/astrbot-plugin-style-learner.git
cd astrbot-plugin-style-learner
pip install -r requirements.txt
```

2. 重启 AstrBot，或在 WebUI 中加载插件。

3. 在 WebUI → 插件配置中，填写 **DeepSeek API Key**（必填）。

## 命令

| 命令 | 用法 | 说明 |
|------|------|------|
| `/sklearn` | `/sklearn <技能名> <文件路径>` | 从 `.jsonl` 文件学习风格 |
| `/sklist` | `/sklist` | 列出所有已学习的技能 |
| `/imitate` | `/imitate <技能名> <消息>` | 以指定风格回复消息 |
| `/skdelete` | `/skdelete <技能名>` | 删除一个技能 |
| `/skinfo` | `/skinfo <技能名>` | 查看技能详情（标签、描述、示例数等） |
| `/skupdate` | `/skupdate <技能名> [新路径]` | 更新技能（不指定路径则用原文件） |

### 风格混合

`/imitate` 支持多风格按权重混合：

```
/imitate 温柔学姐:0.3+沙雕群友:0.7 今天天气真好
```

### 数据格式

对话数据为 `.jsonl` 文件（也支持 `.jsonl.gz`），每行一个 JSON 数组：

```json
["你好呀", "好久不见！最近过得怎么样？"]
["今天天气真好", "是呀，适合出去走走。"]
```

- 每行至少包含 2 个句子（至少一轮对话）
- 支持嵌套数组格式（如 LCCC 数据集）

### 示例：使用 LCCC 数据集

```bash
# 解压
gunzip lccc_base_test.jsonl.gz

# 在聊天中使用
/sklearn 微博风格 D:/datasets/lccc/lccc_base_test.jsonl
/imitate 微博风格 你在干嘛呢
```

## 配置项

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `deepseek_api_key` | (空) | DeepSeek API Key，必填 |
| `deepseek_model` | `deepseek-chat` | 使用的模型名称 |
| `deepseek_base_url` | `https://api.deepseek.com` | API 地址 |
| `max_sample_chars` | `3000` | 学习时单次最大采样字符数 |
